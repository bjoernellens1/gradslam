ICP Tracking

Iterative Closest Point tracking backends.

solve_lm_6x6(A, b, damp=0.0001)[source]

Solve damped normal equations for 6-DOF SE(3) optimization.

Solves: (A^T A + damp * I) x = A^T b for the 6-vector x (Lie algebra twist).

Parameters:

  • A (Tensor) – Constraint matrix of shape [N, 6] (rows = correspondences, cols = DOF).

  • b (Tensor) – Residual vector of shape [N, 1].

  • damp (float) – Damping coefficient (Levenberg-Marquardt lambda). Default: 1e-4.

Returns:

Solution vector x of shape [6, 1].

Return type:

Tensor

Shapes:

  • A: [N, 6]

  • b: [N, 1]

  • Output: [6, 1]

point_to_plane_projective(vertex_live, normal_model, vertex_model)[source]

Compute point-to-plane residuals and Jacobian for projective ICP.

Given live vertices and model vertices/normals, compute the residual and Jacobian matrix for point-to-plane (point-to-surface) alignment.

Parameters:

  • vertex_live (Tensor) – Live vertices in model frame, shape [N, 3].

  • normal_model (Tensor) – Surface normals of model vertices, shape [N, 3].

  • vertex_model (Tensor) – Model vertices (associations), shape [N, 3].

Returns:

  • A: Jacobian matrix, shape [N, 6].

  • b: Residual vector, shape [N, 1].

Return type:

Tuple of

Shapes:

  • vertex_live: [N, 3]

  • normal_model: [N, 3]

  • vertex_model: [N, 3]

  • A: [N, 6]

  • b: [N, 1]
