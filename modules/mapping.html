TSDF volume for incremental dense reconstruction.

Stores signed distance and weight for each voxel, updated via frame-wise fusion.

integrate(depth, intrinsics, T_world_camera, obs_weight=1.0, color=None)[source]

Integrate a depth frame into the TSDF volume.

Parameters:

  • depth (Tensor) – Depth image [H, W] in meters.

  • intrinsics (Tensor) – Camera intrinsics [3, 3] (K matrix).

  • T_world_camera (Tensor) – Camera pose in world frame [4, 4].

  • obs_weight (float) – Weight of this observation. Default: 1.0.

  • color (Tensor | None) – Optional color image [H, W, 3] in [0, 1] or [0, 255].

Return type:

None
