backend_report()[source]

Generate a diagnostic report of the PyTorch/accelerator stack.

Returns:

torch, torch_cuda, torch_hip, cuda_available_api, backend, device_count, device_name.

Return type:

dict with keys

default_device()[source]

Return the default device for tensor operations.

Returns:

torch.device(“cuda”) if available, else torch.device(“cpu”).

Return type:

device

compile_if_requested(fn, *, fullgraph=False)[source]

Conditionally compile a function via torch.compile.

Respects GRADSLAM_COMPILE environment variable (0/1/true/yes). When disabled, returns the original function unchanged.

Parameters:

  • fn (Callable) – Function to compile.

  • fullgraph (bool) – If True, force fullgraph=True in torch.compile (for fixed-shape tests). If False, use fullgraph=False (default, allows dynamic shapes).

Returns:

Compiled function or original function if compilation is disabled.

Return type:

Callable

compile_debug(fn)[source]

Compile with fullgraph=True (useful for debugging graph breaks in tests).

Parameters:

fn (Callable)

Return type:

Callable
